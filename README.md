# Online Book Store

## Project Overview
The Online Book Store is a web-based application that allows users to browse, search, and purchase books online. It provides a seamless shopping experience with secure payments and order tracking.

---

## Problem It Solves
Traditional bookstores are limited by location and working hours. Customers often struggle to find specific books in physical stores. This platform provides:

- 24/7 book access
- Easy search and filtering
- Online secure payment
- Home delivery

---

## Target Users (Personas)

### 1. College Student – Priya
- Age: 20
- Needs affordable academic books

### 2. Working Professional – Rahul
- Age: 30

### 3. Admin – Store Manager
- Manages inventory
- Updates book details

## Vision Statement
To build a user-friendly and scalable online bookstore platform that makes book purchasing easy, secure, and accessible to everyone.
